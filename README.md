LeadFlow CRM
LeadFlow CRM is a simple Lead Management System built using HTML, CSS, JavaScript, Bootstrap, and Chart.js. It helps users manage sales leads efficiently with features like adding, editing, deleting, searching, and filtering leads.

Features
Add new leads
Edit existing leads
Delete leads
Lead status tracking
Search leads
Filter leads by status
Follow-up reminder date
Analytics chart for lead status
Dark mode toggle
Export leads to Excel
Login and logout system
Responsive dashboard UI
Technologies Used
HTML5
CSS3
JavaScript
Bootstrap 5
Chart.js
LocalStorage (for data storage)
